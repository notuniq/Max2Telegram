# Вклад в проект

Спасибо за интерес к проекту! Вы можете помогать двумя способами:

## Предложения
- Используйте [Issues](https://github.com/notuniq/Max2Telegram/issues) → New Issue → Feature Request
- Опишите проблему или улучшение максимально подробно

## Изменения в коде
1. Форкните репозиторий
2. Создайте ветку с именем `feature/имя_фичи`
3. Внесите изменения и протестируйте их
4. Сделайте Pull Request с описанием изменений

## Сообщение об ошибках (bugs)
- Используйте [Issues](https://github.com/<ваш_репозиторий>/issues) → New Issue → Bug Report
- Опишите шаги воспроизведения, ожидаемый и фактический результат
- Прикрепите логи, скриншоты или видео при возможности
